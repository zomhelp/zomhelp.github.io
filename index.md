# འགོ་འཛུགས་པ། 
{: .no_toc }


འཛོམས་ཨེབ་ནི་མཛའ་གྲོགས་རྣམས་ལ་བརྡ་ལན་སྤྲོད་བྱེད་ཀྱི་ཨེབ་ཞིག་རེད། དེ་ཡང་སྤྱོད་བདེ་བ་དང་བཙན་པོ་ཡོད།
